Java 开源项目精选

欢迎大家在项目 issues 区推荐自己认可的 Java 开源项目，让我们共同维护一个优质的 Java 开源项目精选集！

  • GitHub 地址：https://github.com/CodingDocs/awesome-java
  • Gitee 地址：https://gitee.com/SnailClimb/awesome-java
